### Will list all the databases you have on MySQL.

```
show databases;
```

### Creates a database with the name you select.

```
CREATE DATABASE <name>;
```

### Drops or deletes a database that you selected.

```
DROP DATABASE <name>;
```

### To use a database.

```
USE <name>;
```

### To verify the database you are using.

```
SELECT database();
```
